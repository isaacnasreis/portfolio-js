# Bem-vindo ao meu novo Blog!

Decidi evoluir a arquitetura do meu portfólio para incluir uma seção de artigos. O objetivo é compartilhar conhecimentos sobre desenvolvimento Full-Stack, UX/UI e performance.

## Por que Astro?

Astro é um framework focado em conteúdo que entrega **zero JavaScript** por padrão. Isso faz com que sites gerados com Astro sejam incrivelmente rápidos, o que é excelente para SEO.

Aqui está um exemplo de código:

```javascript
const framework = "Astro";
console.log(`Construindo com ${framework}`);
```
